# English AI Tells — Catalog

Reference catalog for the `humanize-prose` skill. Use as a checklist during Step 2 of the workflow.

A "tell" is a stylistic feature that appears disproportionately in LLM-generated text. None of these are wrong on their own — the issue is **frequency** and **stacking**. One "delve" in a 2,000-word essay is fine; three of them with two "leverage"s and a "tapestry" is the giveaway.

## Vocabulary

LLM-favored words and stock phrases. Most have plainer human equivalents.

| Tell | Why it sounds AI | Replacement |
|------|------------------|-------------|
| delve into | overused; rare in spoken English | examine, look at, study, dig into, or just drop |
| leverage (verb) | bizspeak fingerprint | use, draw on, apply |
| robust | catch-all praise word | strong, reliable, sturdy, well-tested |
| comprehensive | filler superlative | full, complete, thorough, or specify what's covered |
| multifaceted | abstract praise | name the facets, or use "complex" |
| holistic | overgeneralized | "looks at the whole" (only if true) or drop |
| intricate | LLM admiration word | detailed, complex, fiddly |
| tapestry (of) | metaphor LLMs reach for | drop the metaphor; describe directly |
| realm (of) | overused metaphor | "area," "field," or drop |
| landscape (of) | same as realm | drop or specify |
| ever-evolving / ever-changing | empty intensifier | drop, or name what changes |
| navigate (challenges) | metaphor cliché | "deal with," "handle," "work through" |
| foster (an environment) | corporate cliché | "build," "support," "create" |
| underscore (verb) | overused for "show" | "show," "highlight," "stress" |
| paradigm | rarely the right word | "approach," "model," or drop |
| plethora | always wrong-feeling | "many," "lots of," or specify |
| myriad | same as plethora | "many" |
| nuanced | LLM hedge praise | name the nuance |
| pivotal | overused | "key," "important," or specify why |
| seamless / seamlessly | marketing-ese | "smooth," "without friction," or drop |
| streamline / streamlined | marketing-ese | "simplify," "speed up" |
| cutting-edge | marketing-ese | "new," "recent," name the technology |
| game-changing / game-changer | marketing-ese | drop or specify the change |
| transformative | empty praise | name the transformation |
| empower / empowering | corporate cliché | "let," "enable," "give … the ability to" |
| testament (a testament to) | overused metaphor | "shows," "proves" |
| underlying (the underlying X) | filler adjective | often deletable |
| in the realm of | filler phrase | "in," "for" |
| at the heart of | metaphor cliché | "central to," "key to" |
| stand the test of time | cliché | drop or be specific about what survived |

## Phrasing & filler intros

Empty hedges that announce a thought is coming without contributing to it.

| Tell | Replacement |
|------|-------------|
| It's important to note that … | drop; lead with the noted thing |
| It's worth mentioning that … | drop |
| It should be noted that … | drop |
| Essentially, … | drop |
| Fundamentally, … | drop unless contrasting with surface |
| In essence, … | drop |
| At its core, … | drop |
| When it comes to X, … | "For X," or drop entirely |
| As mentioned earlier, … | drop or use "(see above)" |
| In today's fast-paced world, … | drop; supply concrete context if needed |
| In today's digital age, … | drop |
| In recent years, … | give a year or specific event |
| It goes without saying that … | drop (if it goes without saying, don't say it) |
| Needless to say, … | drop |
| Of course, … | drop or replace with the actual reason |
| As we all know, … | drop |
| Let's dive in / dive into | drop; just start |
| That being said, … | "Still," "But," "However" |
| Having said that, … | "Still," "But" |
| Not only X but also Y | "X and Y" most of the time |
| It is interesting to note that | drop |

## Structure

Patterns at the sentence and paragraph level.

| Pattern | Why it sounds AI | Fix |
|---------|------------------|-----|
| Tricolon overuse (`X, Y, and Z`) | LLMs default to lists of three | drop to two, or one. Three only when genuinely parallel |
| Reflex bullet lists | LLMs bullet things that should be prose | rewrite as prose if items are continuous; keep bullets only for genuinely scannable lists |
| Sentence-length uniformity | LLM output converges on a middle length | mix short and long deliberately |
| Mirrored clause structure | "Not only … but also …", balanced parallel clauses | break the symmetry |
| Topic sentence then three supports | textbook five-paragraph structure | vary paragraph shape |
| Defining a term then immediately restating it | LLM "explanation" reflex | trust the reader; pick one phrasing |
| Listing pros and cons in symmetric balance | LLM evenhandedness reflex | take a position, or note actual asymmetry |
| Parallel paragraph openers ("First," "Second," "Third,") | LLM scaffolding showing through | drop the labels; let paragraph order carry the structure |

## Punctuation

| Tell | Why it sounds AI | Fix |
|------|------------------|-----|
| em-dash (—) overuse | LLM punctuation tic | use commas, colons, parentheses, or split into two sentences. One em-dash per page is plenty |
| Oxford comma in every list, even short ones | mechanical consistency | fine in formal prose; in casual register, vary |
| Smart quotes everywhere | rendered output marker | fine — but check consistency with the rest of the document |
| Semicolons used as stronger commas | LLM reach for "elegant" punctuation | use a period or "and" |
| Ellipsis as emotional softener (…) | LLM mimic of human hesitation | usually drop; if needed, prefer a period |

## Openers

| Tell | Fix |
|------|-----|
| In today's fast-paced world / digital age / interconnected society | drop, replace with concrete context |
| Imagine a world where … | drop unless this is genuinely speculative fiction |
| Have you ever wondered … | drop; just answer it |
| Picture this: … | drop |
| In an era of [adjective] [noun] | drop |
| The world of X is changing rapidly | drop or be specific |
| X has revolutionized / is revolutionizing Y | usually overstated; specify the change |

## Closers

| Tell | Fix |
|------|-----|
| In conclusion, … | drop; let the last point land |
| Ultimately, … | drop or use only if genuinely concluding |
| All in all, … | drop |
| In summary, … | drop |
| To summarize, … | drop |
| At the end of the day, … | drop |
| As we look to the future, … | drop or be specific |
| Only time will tell. | drop (it's a stock phrase) |
| The possibilities are endless. | drop |
| Hopefully, this article has helped you … | drop |
| Stay tuned for more! | drop unless this is a serial publication |

## Voice & stance

| Tell | Fix |
|------|-----|
| Avoiding contractions in casual writing ("do not" everywhere) | use "don't," "can't," "we'll" where natural |
| Hedging every claim ("It's possible that," "There's a chance that," "It could be said") | take a position when warranted |
| Symmetric "on the one hand … on the other hand" | take a side, or mention only the relevant hand |
| No first-person where it would naturally appear | use "I"/"we" if the writing is genuinely yours |
| Generic "we" in non-collaborative contexts | switch to "I" or drop |

## Frequency markers (count these)

If the input has any of the following at high density, that's a strong AI signal:

- More than 1 em-dash per 200 words
- More than 1 tricolon per paragraph
- More than 1 of {*delve, leverage, robust, comprehensive, multifaceted, intricate, tapestry, realm*} per 500 words
- A bullet list where every bullet is a complete sentence of similar length
- Paragraph lengths within 15% of each other across 5+ paragraphs
- Both a generic opener and a generic closer in the same piece

When you find these, weight your rewrite accordingly.
